# 🏥 AI-Powered Health Management System

A full-stack healthcare management web application developed as a first-year B.Tech Computer Science project.

## 📌 Overview

This application helps users manage their healthcare information through a simple web interface. It includes authentication, health record management, medicine reminders, and AI-assisted health insights.

## ✨ Features

- 🔐 User Authentication
- 📋 Health Record Management
- 💊 Medicine Reminder
- 🤖 AI Health Insights
- 📊 Dashboard

## 🛠️ Technologies Used

- Python
- Flask
- HTML5
- CSS3
- JavaScript
- SQLite

## 🚀 Installation

```bash
git clone https://github.com/YOUR_USERNAME/Smart-Health-Management-System.git
cd Smart-Health-Management-System
pip install -r requirements.txt
python app.py
```
